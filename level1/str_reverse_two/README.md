# Project

Two different algorithms to reverse a given String.

# Description

Your task for today is to write some code that reverses any string.

There are two fundamentally different ways to accomplish this,
one using a for-loop and another using string slicing.

If you can, write both solutions.

# Example:

txt = "Python is fun!"

```bash
$> (Method 1) The reversed string is: "!nuf si nohtyP"
$> (Method 2) The reversed string is: "!nuf si nohtyP"
```

# Prerequisites

Required Libraries: sys
Required Files: No additional files are required.
